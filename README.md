# Simple To-Do List App

A lightweight, responsive, and modern To-Do List web application designed to help users manage their daily tasks efficiently. Built using a clean and standard web stack (HTML5, CSS3, and Vanilla JavaScript), this project features local data persistence and real-time user analytics.

## 🚀 Live Demo

You can check out the live version of the application here:  
👉 [https://noflyzonex.github.io/ToDoList/](https://noflyzonex.github.io/ToDoList/)

---

## ✨ Features

* **Task Management:** Seamlessly add new tasks, toggle their completion status (strike-through style), and delete completed or unwanted items dynamically.
* **Local Storage Persistence:** Your tasks are safe! The app utilizes the browser's `localStorage` API to save your list automatically. Tasks persist even after refreshing the page or closing the browser.
* **Fully Responsive UI:** A modern, clean, neon-inspired user interface designed to look great and work perfectly across all devices (Desktop, Tablets, and Smartphones).
* **Cross-Platform Typography:** Integrated with **Open Sans** via Google Fonts to ensure consistent, elegant, and professional text rendering on both iOS/Android mobile browsers and Desktop systems.
* **Real-time Analytics:** Integrated with Google Analytics 4 (GA4) to privately track page views, user sessions, and interaction events in real time.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Custom variables, flexbox, and absolute positioning)
* **Logic:** Vanilla JavaScript (ES6+, DOM manipulation, Event listeners, and Web Storage API)
* **Fonts & Icons:** Google Fonts (Open Sans), Font Awesome Icons (v6)
* **Analytics:** Google Analytics (gtag.js)
* **Hosting:** GitHub Pages

---

## 💻 Code Overview & Structure

The project structure is minimal and optimized for fast loading:

* `index.html`: Contains the structural markup, Google Analytics tracking scripts, and links to external styles/icons.
* `style.css`: Handles the global layout, the responsive form container, custom input states, typography imports, and hover/click effects.
* `index.js`: Manages the application logic, dynamic DOM injection for new tasks, interactive event handlers for toggling/deleting, and local storage updates.
